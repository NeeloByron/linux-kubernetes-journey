## 🎯 Course Objectives
- Package applications into containers and deploy them to a cluster.
- Work with essential Kubernetes objects: Pods, Deployments, Services, ConfigMaps, Secrets, Ingress, and PersistentVolumes.
- Explore networking, storage, and security in Kubernetes.
- Use kubectl effectively to interact with the cluster.
- Gain hands‑on experience to confidently approach the CKAD exam.

## ✅ Prerequisites
- Basic Linux command‑line skills.
- Familiarity with containers (Docker) is helpful but not mandatory.
- A Kubernetes cluster (local like Minikube or Kind, or a cloud‑based cluster like AKS/EKS/GKE) – the course provides guidance.

## 🚀 How to Use This Folder
1. **As I progress through the course**, I'll add my notes in the `notes/` subfolder, organised by module (e.g., `notes/module1-basics.md`).
2. **Labs** from the course will be saved in `labs/` – each lab can be a subfolder or a single YAML file with comments.
3. **Projects** are larger personal exercises; each project gets its own subfolder under `projects/` with its own README and source files.
4. **Diagrams** (if any) go into `diagrams/` – useful for visualising networking or architecture.
5. I will commit regularly to track my progress and keep everything backed up.
